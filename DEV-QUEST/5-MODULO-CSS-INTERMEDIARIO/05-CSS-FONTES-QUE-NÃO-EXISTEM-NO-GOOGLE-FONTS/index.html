Baixando fontes customizadas

primeiro basta ir em um site de fontes como o "dafonte" onde peguei essa fonte acima cada browser usa um modelo de fonte mas os mais usados são os tipos "otf" e "ttf", enfim basta baixa-las e criar uma pasta para elas depois no css colocar a tag "@font-face { font-family: ; src: url(); }" onde em font-family entre '' aspas comuns se você coloca o nome que quer usr para chamar a fonte e em url usa ./ para linkar a pasta caso tenha os dois tipos basta por um vírgula e chamar a outra, exemplos no arquivo css.
