Divs e spans

uma tag "div" é ultilizada para agrupar elementos ou estilizar esses elementos, só deve ser usada caso não exista uma tag mais semantica.

A tag "span" serve para selecionarmos elementos especificos dentro de um valor para um elemento e não quebrar a linha.
