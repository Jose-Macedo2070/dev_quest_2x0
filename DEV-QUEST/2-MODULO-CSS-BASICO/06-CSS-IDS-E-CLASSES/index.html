Ids e classes

O atributo "id" serve para darmos nome e identificarmos certas tags em nosso codigo mas não é tão usual pois não dá para termos mais de um "id" igual a outro. não pode ser ultilizado um igual na mesma página. no css para chamarmos o "id" usamos "#". Em laranja temos o "id"

O atributo "class" é mais usual pois podemos colocar mais de uma classe igual em outros elementos. para chamarmos ele no css ultilizamos ".". podemos ainda aplicar mais de uma classe para um elemento. Em ciano temos a classe.
