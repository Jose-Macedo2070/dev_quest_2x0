(que pega todos os arquivos) de staged para unmodified: git commit -m caso voce modifique um arquivo ele vai sair de de unmodified para modified ai precisa fazer o processo novamente de git add e git commit -m o ultimo passo seria o git push que seria levar os arquivos para o github: git push -u origin main caso queira pegar os arquivos para usar em outra máquina deve usar o git clone e o código da pasta que quer, lembrando de abrir o bash na reparição certa como area de trabalho e tal. ex: git clone https://github.com/Jose-Macedo2070/AULA-TABELA.git git pull origin main --rebase: serve para baixar as atualizações da pasta git que esteja na máquina que foi feita o clone, para não dar conflito entre tudo novo a ser baixado.
