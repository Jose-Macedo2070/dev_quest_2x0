Input text

gera um input de text para enviarmos textos

nele usa-se os atributos "type" sendo o principal e definindo o tipo do input, o atributo "name" que define o nome do input, ela é ultilizada paara pegar o campo tratando os dados do texto, o "value" passa o valor do input, o id ajuda quando criamos um "label" e o "placeholder" que ajuda colocando textos informativos que são sobreescritos com informações novas.
