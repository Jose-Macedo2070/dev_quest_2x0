Formulário são todas as listas feitas para pegar dados inputados com a tag "input" e seus atributos por um usuário e enviar para algum servidor de backend

Temos dois tipos de método para colocar em prática os dados de um formulário que entram junto do atributo "method" que seriam o "get" e o "post" onde o get faz a colocação dos dados aparecerem na aba de pesquisa junto com o "name" que foi colocado na tag form. Já o Post seria uma form mais segura ligada a um banco de dados de backend.
