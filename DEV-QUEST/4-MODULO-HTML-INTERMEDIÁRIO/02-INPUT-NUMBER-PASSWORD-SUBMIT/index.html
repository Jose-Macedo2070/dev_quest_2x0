Inputs Number, Password, Submit

o input "number" gera uma caixa de texto onde só é permitido digitar números, o input "password" gera um caixa de texto que oculta os caracteres como se fosse uma senha mesmo

a tag de "label" colocada serve para nomear as ciaxas de texto junto do atributo"for" que é igado diretamente ao atributo "id" ou seja o nome que tiver no atributo "for" tem que ser o mesmo do "id" nesse caso





Para o input "submit" funcionar ele precisa estar dentro da tag "form" que vai gerar uma ação de envio de formulários
