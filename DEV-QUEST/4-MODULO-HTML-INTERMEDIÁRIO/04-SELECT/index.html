SELECT

A tag "select é utilizada para criar opções selecionáveis para a pessoa que está tendo o acesso ao site, dentro dela criamos uma tag "option" para criar as opções e junto a essa tag passamos o atributo "value" pois é ele que vai aparcecer e passar o valor da opção escolhida. o que vai ser visto é o que está dentro do "option" mas o que passa o valor pro sistema é o que está no "value"
